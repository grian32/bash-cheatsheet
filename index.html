Bash Cheatsheet

A small and to the point cheatsheet for bash scripting. This is currently WIP.


Shebang

These are used to indicate the interpreter that will be used for execution. 

#!/bin/bash

Strings

Escaped Strings

Escaped strings are marked by double quotes and process escapes. These also have interpolation. 

"hi\nbye" # \n escapes to a new line

# also used for multiline strings
"hi
bye"

#interpolation
"${var}"

Unescaped Strings

Unescaped strings are marked by single quotes and treat escapes as raw characters 

'hi\nbye' # hi\nbye, literal \ and n, backslash does not act as an escape character

# no interpolation
'${var}' # literal characters, doesn't interpole the variable

Variable Declaration

Spaces aren't allowed between the variable name and the contents. 

hi="hi"

To assign the output of a command to a variable:

hi_contents=$(cat hi.txt)

Basic Commands

echo

This command is a basic print command. 
            echo "Hello, World!"
echo "${variable_name_here}"
echo "hello ${variable_name_here}"

cat

This command prints the contents of the given file. 
cat file_name_here

ls

This lists all the files and directories in the given directory. 
ls /absolute/path/to/dir/  
ls relative/path/to/directory
# also supports wildcards and that kind of thing
ls *.txt

touch

This creates a empty file. 
touch file_name.txt

Control Flow

If Statements

if ["$somevar" == "hi"]
then
    echo "hi"
fi

# multiple conditions in one if
if ["$somevar" == "hi"] || ["somevar" != "bye"] # Uses boolean operators.
then
    echo "hi but not bye"
fi

For Loops 

for i in {1..5}
do
    echo "Your number is $i"
done

# traditional for loop
for ((i = 1; i <= 5; i++))
do
    echo "Your number is $i"
done
