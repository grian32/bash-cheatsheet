Bash Cheatsheet

A small and to the point cheatsheet for bash scripting This is currently WIP.

Shebang 

#!/bin/bash

Variable Declaration

It's required that there are no spaces between the variable name and its contents. 

hi="hi"

Assigning the output of a command to a variable: 
hi_contents=$(cat hi.txt)

Basic Commands

echo

This command is the basic print command 
echo "Hello, World!"
echo "${variable_name_here}"
echo "hello ${variable_name_here}"

cat

This command prints the contents of the given file 
cat file_name_here

ls

Lists all the files and directories in the given directory 
ls /absolute/path/to/dir/             
ls relative/path/to/directory
# also supports wildcards and that kind of thing
ls *.tx

Control Flow

If Statements

if ["$somevar" == "hi"]
then
    echo "hi"
fi

# multiple conditions in one if
if ["$somevar" == "hi"] || ["somevar" != "bye"]
then
    echo "hi but not bye"
fi

For Loops 

for i in {1..5}
do
    echo "$i"
done

# traditional for loop
for ((i = 1; i <= 5; i++))
do
    echo "$1"
done
